# demo — примеры моделирования radar3d

Серия **связанных** примеров сквозного тракта. Наполняется **по шагам** — задачи ставит Alex,
перечень меняется. Каждый следующий пример переиспользует предыдущий (сигнал / сцену / стенд).

## Структура

| путь | назначение |
|------|-----------|
| `demo/core/`     | общий стенд (Template Method-раннер, банк сцен, writer, report) — по мере задач |
| `demo/graphics/` | вывод PNG, подкаталог на пример (в `.gitignore`) |
| `demo/tests/`    | приёмка примеров через `common.runner.TestRunner` (🚫 pytest) |

## Примеры (ориентир, меняется)

1. АМ на одной временно́й оси.
2. Сигнал из (1) → апертура `nx×ny×N`, обработка 3FFT.
3. + заградительная помеха (barrage).
4. ЛЧМ через тот же стенд.
5. FM-m код: истинная цель vs ретранслятор → арбитр гл.5.

Детали-ориентир: [`MemoryBank/specs/demo_examples_2026-07-18.md`](../MemoryBank/specs/demo_examples_2026-07-18.md).

## Правила

- Пакет строчный (`demo`, не `Demo`) — Linux ФС регистрозависима.
- Имена папок примеров без пробелов (`ex1_am_line`, …); красивые названия — здесь и в докстрингах.
- Не изобретать генераторы/классификаторы/арбитры — только связывать готовые из `core/*`.
