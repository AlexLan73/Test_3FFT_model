## Принципы

- **Инверсия зависимостей (DIP)**: `SimulationController` зависит от абстракций
  (`RadarModel`, `SceneBuilder`, `DataContext`), связывание — в `main.py`.
- **Open/Closed**: новая помеха / окно / визуализатор / модель добавляются
  подклассом + регистрацией, без правки существующего тракта.
- **Liskov**: `Cnn3DClassifier` заменяет `RuleBasedClassifier` по тому же
  интерфейсу `CubeClassifier`.
- **Чистота**: источники не мутируют вход, конфиги `frozen`, прогон детерминирован
  (`seed`).
