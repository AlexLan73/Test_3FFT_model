## Апертура i×j / 2ⁿ и полный конвейер

Апертура — **i×j** (не обязательно квадрат), каждая ось паддится независимо до
ближайшей степени двойки (`ArrayConfig.padded_shape()`). Полный когнитивный
конвейер: фронтенд (2 FFT / скользящий 3D-FFT) → куб → **токенизатор** (гл.4/4-бис)
→ **арбитр** (гл.5) → **целеуказание** (гл.8) → **трекинг**, плюс параллельная
ветка anti-barrage. Детали формулы — [`Doc/Patent/00_КОНЦЕПЦИЯ_ixj_2n.md`](../Patent/00_КОНЦЕПЦИЯ_ixj_2n.md),
обзор конвейера — [`README.md`](../../README.md) корня репозитория.

## Принципы

- **Инверсия зависимостей (DIP)**: `SimulationController`/`CognitiveCycle`
  зависят от абстракций (`RadarModel`, `Arbiter`, `Targeting`, `Tracker`,
  `DataContext`), связывание — в `main.py`/`demo_*.py`.
- **Open/Closed**: новая помеха / окно / визуализатор / фронтенд / арбитр /
  трекер добавляются подклассом + регистрацией, без правки существующего тракта.
- **Liskov**: `Cnn3DClassifier` заменяет `RuleBasedClassifier` (`CubeClassifier`);
  `CodeArbiter`/`EdgeArbiter` взаимозаменяемы (`Arbiter`);
  `RobustMvdrNuller` заменяет `SubspaceNuller`.
- **Чистота**: источники не мутируют вход, конфиги `frozen`, прогон детерминирован
  (`seed`).
