# Карта проекта: способ, устройство и статья (ECCM / унифицированный объёмный примитив)

## Стержень
Одно **ядро** (объёмный токенизатор → ИИ-гейт → причинностный арбитр → агильный опрос FM-m) со **сменными интерфейсами** под тип сигнала: ЛЧМ — точность по дальности, AM — быстрое присутствие. Токен на объёме (куб `16×16×L`) — универсальный интерфейс; N переменна; плоскость = частный случай `L=1`.

## Статья (набело, по главам — отдельные файлы)
- `glava0_arhitektura` — первый лист: архитектура (ядро + сменные интерфейсы)

- `glava3_dva_ffta` — **Интерфейс ЛЧМ**: два FFT (дальностный + угловой 16×16)
- `glava4_tokenizaciya` — токенизация угловой карты (признаки, гейт, токен)
- `glava4bis_obyomnyj_primitiv` — **Ядро**: объёмный примитив (3D-FFT, AM-фронтенд, OS-CFAR-3D, объект↔RFI, бюджет MI100)
