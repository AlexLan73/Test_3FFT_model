# Название изобретения

Способ распознавания истинной цели на фоне самоприкрывающей ретрансляционной помехи и устройство для его осуществления.

# Область техники

Изобретение относится к радиолокации, а именно к защите радиолокационных станций с активной фазированной антенной решёткой (АФАР) от активных ретрансляционных (DRFM) самоприкрывающих помех, и может использоваться для распознавания истинной цели среди ложных отметок и выдачи целеуказания.

# Уровень техники

Известен способ подавления активной помехи (прототип RU2549375C1), основанный на приёме излучения источника помех по двум каналам — основному и вспомогательному — и вычислении корреляции между ними для компенсации помехи в главном луче. Прототип рассчитан на геометрию отдельного постановщика помех и не содержит структурного различителя «истинная цель / ложная цель»: при самоприкрытии, когда помеха излучается с носителя самой цели, отклик ретранслятора структурно не отличается от отражения цели, и прототип задачу не решает.

Известны также нейросетевые классификаторы радиолокационных данных на трёхмерном массиве «дальность–угол–скорость» (US10962637B2, US12181562), когнитивные планировщики распределения радиолокационного ресурса (US11747438), формирование азимут-угломестной картины преобразованием Фурье по бинам дальности (EP0097490A2) и GPU-ускоренная обработка радиолокационных данных (CN101441271A). Каждое из этих решений по отдельности не обеспечивает распознавания истинной цели при самоприкрытии.

# Раскрытие сущности изобретения

**Решаемая задача** — надёжное распознавание истинной цели на фоне самоприкрывающей ретрансляционной помехи (гребёнка ложных целей, размытие спектра, заградка, сторонний источник) при ограниченной латентности обработки.

**Технический результат** — повышение достоверности распознавания истинной цели и снижение вычислительных затрат за счёт сведения большого объёма данных к компактной матрице токенов и применения причинностного физического арбитра.

Технический результат достигается **совокупностью признаков**, работающих за каждый рабочий такт:

1. излучение широкого линейно-частотно-модулированного (ЛЧМ) зондирующего сигнала и формирование объёма «апертура×апертура×дальность» размерности 16×16×N (N — переменная величина такта) **двумя раздельными преобразованиями Фурье** — глобальным дальностным и поячеечным угловым 16×16;
2. замена полной угловой карты каждого среза дальности **структурным токеном** из ограниченного числа наиболее ярких угловых пиков и вектора **инвариантных к расщеплению (страддлу) признаков** распределения энергии;
3. классификация срезов нейросетевым **гейтом** при вынесении окончательной метки «истинная/ложная цель» **причинностным физическим арбитром** — по правилу ближнего по дальности члена причинностной группы (передний край, τ ≥ 0) и/или согласования отклика с текущим кодом зондирования;
4. формирование целеуказания по матрице токенов и излучение в выбранные площади **многолучевого агильного фазоманипулированного зонда FM-m** с кодом переменной длины 2ⁿ, меняющимся от такта к такту.

Ни один из признаков 1–4 по отдельности не обеспечивает технический результат; он достигается их совокупностью.

# Перечень фигур

- Фиг. 1 — схема формирования объёма 16×16×N двумя раздельными преобразованиями Фурье.
- Фиг. 2 — угловая карта и вектор инвариантных признаков для трёх классов среза.
- Фиг. 3 — устойчивость признаков к расщеплению (страддлу).
- Фиг. 4 — структурный токен и сборка токенов по дальности.

*(Чертежи прилагаются отдельно; для целей формулы не обязательны.)*

# Осуществление изобретения

**Тракт приёма.** АФАР → АЦП → гетеродин-дечирп на ПЛИС (stretch-processing) → комплексные IQ. Дечирп превращает задержку цели в постоянную частоту биений (`f_b = μ·2R/c`, `μ = ΔF/T_c`); за такт формируется N комплексных отсчётов, N переменна.

**Формирование объёма (два FFT).** Глобальным дальностным преобразованием Фурье по оси быстрого времени биения переводятся в дальность (ось Z становится дальностью, N бинов). Затем на каждом бине дальности выполняется поячеечное угловое преобразование 16×16 на матричных (тензорных) блоках; магнитуда `|·|²` с fftshift берётся только после углового преобразования. Разрешение по дальности `Δr = c/(2·ΔF)`; угловая шкала `sinθ = k/8` при `d = λ/2`.

**Токенизация.** Полная угловая карта (256 = 16×16 ячеек) заменяется структурным токеном: массив до пяти наиболее ярких пиков `(k_x, k_y, амплитуда, кромка)` и вектор инвариантных к расщеплению признаков распределения энергии (пиковое отношение `PR = (Σp)²/Σp²`, мера разреженности Хойера, доля главного лепестка 3×3, интегральное отношение лепестков). Формирование, токенизация и классификация выполняются целиком в быстрой внутрикристальной памяти рабочей группы GPU без выгрузки во внешнюю память.

**Гейт и арбитр.** Классификатор (малая нейросеть 6→16→3) относит срез к классу состояния (шум/собранный/размазанный), выполняя функцию гейта и приоритизатора. Окончательную метку «истинная/ложная цель» выносит причинностный арбитр: истинная цель — ближний по дальности член причинностной группы (ретранслятор причинно только задерживает, `τ ≥ 0`, поэтому ложные отметки всегда дальше), и/или отклик, согласованный с текущим кодом зондирования (свежий непредсказуемый код не может быть подделан ретранслятором). Область применения — самозащитная (самоприкрывающая) помеха.

**Коррелятор FM-m.** Точечный многолучевой опрос реализуется корреляцией через БПФ: `corr = IFFT(conj(FFT(ref))·FFT(inp))`, `O(N log N)`, для S лучей против K гипотез задержки текущего кода одновременно, в двух потоках GPU с батчингом по свободной памяти; на выходе IFFT ядро токенизации извлекает 3–5 наибольших значений отклика.

**Глубокий добор (L3).** На трудных кандидатах с нерегулярной структурой применяется резидентная последовательностная нейросеть (LSTM/GRU) над потоком токенов области интереса, уточняющая угол и тонкую дальность; окончательное подтверждение — код-корреляцией FM-m.

**Целеуказание.** По матрице токенов формируется целеуказание (номер бина дальности, тонкая дальность интерполяцией пика, углы) и излучается многолучевой агильный зонд FM-m в интересуемые площади. Когнитивная петля: обнаружил широко (ЛЧМ) → дал точки (токены) → зондировал точечно (FM-m) → пауза → соседние площади.

**Устойчивость.** Причинность неподделываема (враг не ответит кодом, которого не слышал, и не поставит ложную цель ближе истинной); анти-затопление ограничивает латентность потолком кандидатов и приоритетом по инвариантному скору; устойчивое обнаружение — OS-CFAR по угловой координате; заградительный шум обнаруживается как класс «размазанный» и не подтверждается код-корреляцией.

# Формула изобретения

**1.** Способ распознавания истинной цели на фоне самоприкрывающей ретрансляционной помехи и целеуказания в моностатическом радаре с активной фазированной антенной решёткой, включающий излучение зондирующего сигнала, приём отражённого сигнала элементами решётки, преобразование Фурье принятого сигнала и обнаружение целей, **отличающийся тем, что** за каждый рабочий такт: (а) излучают широкий линейно-частотно-модулированный сигнал, после гетеродинного сжатия получают N комплексных отсчётов при переменной N и формируют массив «апертура×апертура×дальность» размерности 16×16×N двумя раздельными преобразованиями Фурье — глобальным дальностным и поячеечным угловым 16×16; (б) для каждого среза дальности заменяют полную угловую карту структурным токеном из ограниченного числа наиболее ярких угловых пиков и вектора инвариантных к расщеплению признаков распределения энергии; (в) классификатором относят срез к классу состояния, причём классификатор выполняет функцию гейта, а принадлежность отклика истинной либо ложной цели устанавливают по правилу ближнего по дальности члена причинностной группы (τ ≥ 0) и/или согласования отклика с текущим кодом зондирования; (г) по совокупности токенов формируют целеуказание и излучают в выбранное направление точечный фазоманипулированный зондирующий сигнал с агильным кодом переменной длины 2ⁿ, дополняемой нулями до размера преобразования Фурье, причём код меняется от такта к такту.

**2.** Способ по п.1, отличающийся тем, что вектор признаков включает пиковое отношение `PR = (Σp)²/Σp²`, меру разреженности Хойера, долю энергии главного лепестка в окне 3×3 и интегральное отношение главного лепестка к боковым.

**3.** Способ по п.1, отличающийся тем, что структурный токен содержит до пяти наиболее ярких пиков, каждый из которых задан угловыми координатами, амплитудой и признаком кромки; массив пиков упорядочен по углу, а сборка токенов ведётся по дальности.

**4.** Способ по п.1, отличающийся тем, что причинностный арбитр применяют в области самозащитной помехи, а уводящую помеху на ином носителе разделяют по свежести кода зондирования и угловому разнесению.

**5.** Способ по п.1, отличающийся тем, что корреляцию с кодом зондирования выполняют через быстрое преобразование Фурье по k сопряжённым опорам текущего кода с батчингом по свободной памяти графического процессора.

**6.** Способ по п.5, отличающийся тем, что на выходе обратного преобразования Фурье коррелятора формируют вторичный набор токенов из 3–5 наибольших значений отклика.

**7.** Способ по п.1, отличающийся тем, что на трудных кандидатах с нерегулярной структурой дополнительно применяют резидентную последовательностную нейронную сеть (LSTM/GRU) над потоком токенов области интереса, уточняющую угол и тонкую дальность, с подтверждением код-корреляцией; альтернативно — объёмную нейронную сеть на сыром патче 16×16×(32/64/256).

**8.** Способ по п.1, отличающийся тем, что устойчивое обнаружение на угловой карте выполняют посредством CFAR порядковой статистики по угловой координате.

**9.** Способ по п.1, отличающийся тем, что формирование углового среза, свёртку среза в токен и классификацию выполняют целиком в быстрой внутрикристальной памяти рабочей группы графического процессора, а угловое преобразование 16×16 — на матричных (тензорных) вычислительных блоках.

**10.** Устройство распознавания истинной цели и целеуказания, содержащее активную фазированную антенную решётку, аналого-цифровой тракт с гетеродинным сжатием и вычислитель на графическом процессоре, **отличающееся тем, что** вычислитель выполнен с возможностью формирования массива 16×16×N двумя раздельными преобразованиями Фурье, свёртки каждого среза дальности в структурный токен из инвариантных к расщеплению признаков и причинностного арбитража «истинная/ложная цель» по правилу ближнего края и/или согласования с текущим кодом, и содержащее формирователь агильного фазоманипулированного зондирующего сигнала переменной длины 2ⁿ, управляемый по матрице токенов для целеуказания.

**11.** Устройство по п.10, отличающееся тем, что формирователь строит код переменной длины от 2⁸ до 2²⁰, меняющийся от такта к такту.

**12.** Устройство по п.10, отличающееся тем, что вычислитель реализует коррелятор на библиотеке hipFFT/ROCm с двухпоточным конвейером и батчингом по свободной памяти.

**13.** Устройство по п.10, отличающееся тем, что вычислитель выполняет угловое преобразование 16×16 и классификацию в памяти рабочей группы на матричных вычислительных блоках.

**14.** Устройство по п.10, отличающееся тем, что содержит канал широкого ЛЧМ-обзора для грубого целеуказания, гейтящего точечный фазоманипулированный зонд.

# Реферат

Способ и устройство распознавания истинной цели на фоне самоприкрывающей ретрансляционной (DRFM) помехи в моностатическом радаре с АФАР. За каждый такт широкий ЛЧМ-сигнал формирует объём «апертура×апертура×дальность» 16×16×N двумя раздельными преобразованиями Фурье; каждый срез дальности сворачивается в компактный структурный токен из наиболее ярких угловых пиков и инвариантных к расщеплению признаков; нейросетевой гейт классифицирует срезы, а окончательную метку «истинная/ложная цель» выносит причинностный арбитр по правилу ближнего края (τ ≥ 0) и/или согласования с текущим кодом; по матрице токенов формируется целеуказание и излучается многолучевой агильный зонд FM-m с кодом переменной длины. Достигается повышение достоверности распознавания при самоприкрытии и снижение вычислительных затрат.
