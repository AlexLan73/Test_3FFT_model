3. Блок запись данных на ГПУ k-антенн по l комплексных данныъ по два вещественных числа, k*l данных.
4. Управления потоком данных и алгоритмами расположенных на ГПУ, и передача их в сжатом виде, ввиде разряженной матрими из токенов в блок 1, 
так же получает управляющие команды от блока 1 с каким пипом сигнала сейчас работаем и с какими параметрами.  
5. Блок обрабатывает данные относяциеся к ЛЧМ, производит FFT кажого луча параллельно после дечирпирования после этого у нас получается ось ...(допиши как мы ее называем).
6. Блок для обработки АМ сигнала, переформианирование данных из k*l в набор X*Y*Z с перекрытием данных по линии Z
7. Блок обрабатывает ФМ сигнал, производит FFT кажого луча параллельно и в памяти ГПУ записывается спектры входных лучей k*l спектров.
8. Блок формирует в памяти ГПУ данные из k*l в набор i*j*l, где данные из антенн формируются в прямоугольник (квадрат) положим 16х16 на одно значения по дальности.
9. Блок делает 2D FFT на кадый шаг по дальности.
10. Разбор полученного спеетра из 2D FFT по четырем признакам, и формирование массива структур с набором токенов на базе предобученной ИИ для кластиризации.
11. Блок формирует в памяти ГПУ данные из k*l в набор квадрата с размерностью i*j*n где данные из антенн формируются в куб данных к примеру 16х16х16 на одно значения по дальности 
с перекрытием по дальнлсти 8.
12. Блок делает 3D FFT на каждый блок по дальности
13. Разбор полученного спеетра  из 3D FFT по четырем признакам, и формирование массива структур с набором токенов  на базе кластиризации ИИ.
14. Разбор прореженной матрицы токенов на определениеодиночная цель, шум, заградительный сигнал или гребенка из помех. На выходе мы получаем массив структур набора токенов, об одиночных объектов
углы и дальноть по ним, или сигнал для более глубогоро иследования.
15. Блок реализует объемный анализ прореженнлго иаммиса стркутур к  примеру формата 16х16х32/64/..256 с использованием предобученной ИИ LSTM. На выходе мы получаем массив структур набора токенов, 
если одиночный объект углы и дальноть, и характеристика помехи заградительный шум или гребенка.
16. Формируем сопряженный опорный сигнал m-последовательности на количество точек принимаемых от ФМ сигнала. 
17. Формируются вектора циклическиъ сдвигов для формирования набора корреляционных функций. 
18. Считаем спектр этих векторов.
19. Блок перемножения опорного сигнала и исслодоваемого.
20. Блок вычисления обратного преобразования фурье IFFT.
21. Блок в токинезации на выходе получаем массив структур с информацией об обекте.
