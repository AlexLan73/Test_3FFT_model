# CN101441271A - SAR real time imaging processing device based on GPU

## Patent Information
- **Patent Number**: CN101441271A
- **Publication Date**: 2009-05-27
- **Assignee**: Space Star Technology Co Ltd

## Abstract

The GPU-based SAR real-time imaging processing equipment mainly includes the SAR raw data acquisition module, which converts the raw data in the form of high-speed serial code stream into low-speed parallel code stream of ECL level, and then converts it into LVTTL level code stream and sends it to FPGA. Then use the controller in the FPGA to store the data in the cache, and then upload it to the server through the PCI bus; the server sends the received data to the disk array and the GPU-based real-time imaging module to calculate the Doppler center frequency and adjust The frequency parameters are sent to the GPU-based real-time imaging module, and the results are obtained in real time for real-time display; the GPU-based real-time imaging module uses the GPU for imaging processing according to the received data, Doppler center frequency and modulation frequency parameters, and the imaging The processed data is sent to the server; the disk array receives the data and records it in real time. The invention has the advantages of real-time imaging, fast processing speed, simple structure, easy expansion, real-time recording of original SAR data and stable and reliable system.

## Description

The GPU-based SAR real-time imaging processing equipment represents an advancement in synthetic aperture radar (SAR) technology, utilizing GPU acceleration for real-time image processing. The system integrates several key components:

### System Architecture
- SAR raw data acquisition module
- Data conversion systems (ECL to LVTTL level conversion)
- FPGA-based controller for data management
- GPU-based real-time imaging module
- Disk array for data recording

### Key Technical Features
- Real-time processing capability
- High-speed serial to low-speed parallel data stream conversion
- Integration with cache management through FPGA
- GPU-accelerated Doppler processing
- Frequency parameter adjustment and modulation

### Advantages
- Real-time imaging capability
- Fast processing speed
- Simple structure and easy expansion
- Real-time recording of original SAR data
- Stable and reliable system operation

### Technical Innovation
The key innovation is the integration of GPU acceleration for SAR image processing, enabling real-time generation of synthetic aperture radar imagery. The system architecture demonstrates the feasibility of hardware-accelerated SAR processing through the use of General Purpose Graphics Processing Units (GPUs), representing a significant advancement from earlier CPU-based approaches.
