## Abstract

A radar data processing device includes at least one analog-to-digital converter (ADC) configured to digitize a plurality of input signals, wherein each input signal includes radar chirp and radar chirp reflection information received at one of a plurality of receiver antennas. The radar data processing device also includes Fast Fourier Transform (FFT) logic configured to generate FFT output samples based on each digitized input signal, wherein at least some of the generated FFT output samples are across antenna FFT output samples associated with at least two of the plurality of receiver antennas. The radar data processing device also includes a processor configured to determine a plurality of object parameters based on at least some of the generated FFT output samples, wherein the processor uses a neural network classifier trained to provide a confidence metric for at least one of the plurality of object parameters.

## Description

### Background

In a Frequency Modulated Continuous Wave (FMCW) radar system, a sinusoid signal whose frequency increases linearly over time, also known as a "chirp", is transmitted, and one or more objects around the FMCW radar system reflect the transmitted chirp. A sequence of equispaced chirps are transmitted in a unit called a frame. The FMCW radar system processes chirps and related reflections to analyze characteristics of the object. Exemplary characteristics include the range of an object relative to the radar origin, the velocity of an object relative to the radar origin, and the angle of an object relative to a radar origin.

Each chirp and related reflections are signals with varying amplitude as a function of time. One example technique to analyze characteristics of the object(s) involves obtaining Fast Fourier Transform (FFT) output samples of a plurality of input signals, where each input signal includes a chirp and related reflections. By analyzing the FFT output samples, FMCW radar systems have been used to identify presence of an object, range of an object relative to a radar origin, velocity of an object relative to a radar origin, and angle of an object relative to a radar origin.

### Summary and Technical Solution

In accordance with the disclosure, a radar data processing device comprises at least one analog-to-digital converter (ADC) configured to digitize a plurality of input signals, wherein each input signal includes radar chirp and radar chirp reflection information. The radar data processing device also comprises Fast Fourier Transform (FFT) logic configured to generate FFT output samples based on each digitized input signal. The radar data processing device also comprises a processor configured to determine a plurality of object parameters based on the generated FFT output samples, wherein the processor uses a neural network classifier trained to provide a confidence metric for at least one of the plurality of object parameters.

The neural network classifier (also referred to as artificial neural network or ANN) is trained to provide an output for new inputs based on one or more training inputs. Over time, the training for the function can be updated as additional training inputs become available. In different examples, confidence metrics provided by the neural network classifier are used for object detection, object tracking (e.g., tracking object position, velocity, and/or angle over time), and/or updating a constant false alarm rate (CFAR) detection threshold.

### Key Technical Features

**Confidence Metrics:** A confidence metric refers to a probability value that varies between two thresholds (example probability value range is 0.0-1.0 or 0%-100%).

**Object Parameters:** Example object parameters include:
- Object range relative to a radar origin
- Object velocity relative to a radar origin  
- Object angle relative to a radar origin (azimuth/elevation or direction of arrival)

**Neural Network Application:** The neural network classifier is configured to provide a confidence metric for an object parameter based on Fast-Fourier Transform (FFT) output samples obtained for each of a plurality of input signals. The plurality of input signals are obtained from different antennas to support object angle analysis relative to the radar origin.

**Advantage Over Prior Art:** By using a neural network classifier to provide confidence metric values, detection and/or tracking of smaller objects and/or multiple objects is improved compared to traditional radar data processing techniques. The confidence metrics are not based solely on signal-to-noise ratio (SNR).

### Exemplary Processing Operations

The radar data processing device performs:

1. **Range Analysis:** Uses FFT output samples for individual chirps to determine the range of one or more objects relative to a radar origin.

2. **Doppler Analysis:** Analyzes each range bin across chirps to determine the velocity of one or more objects relative to a radar origin.

3. **Angle Analysis:** Uses FFT output samples from multiple receiver antennas to determine the angle (azimuth/elevation or direction of arrival) of one or more objects relative to a radar origin.

4. **CFAR Detection:** Constant False Alarm Rate detection is used to detect the presence of objects, with confidence metrics from the ANN classifier used to adjust CFAR thresholds for improved sensitivity.

### Detailed Processing Techniques

**Across-Antenna 2D-FFT Operations:** When FFT output samples from multiple receiver antennas are available, across antenna 2D-FFT operations can be performed resulting in across antenna FFT output samples for different elevations and azimuths. ANN classifier operations provide confidence metrics based on signal strength cleanliness analysis - if spectrum indicates signal strength is concentrated around limited elevation-azimuth bins, confidence is high; if scattered around many bins, confidence is low.

**Signal Strength Coherency Analysis:** The ANN classifier can identify presence of objects by analyzing signal strength and coherency information. For example, if signal strength is coherent (single peak) across multiple range indices, object presence is likely. If signal strength is incoherent (multiple peaks), object presence is unlikely.

**Threshold Adjustment:** Confidence metrics can be used to dynamically adjust CFAR detection thresholds - decreasing threshold increases detection sensitivity (detecting more objects at cost of more false positives), while increasing threshold decreases sensitivity.
