## 1. Архитектура победителя (глубокий анализ, проверено кодом)

```
Этап 1: ГДЕ — мультимасштабный СТФТ (окно W × Хэмминг + W нулей, шаг W/2)
        W=64 → +18 дБ/бин → длинный импульс (>=64 отс) виден на −6 дБ  ← идея Alex
        W=16 → +12 дБ/бин → короткие импульсы, точнее по времени
        Правило: полный выигрыш только при длительности >= W (короче — теряем).
Этап 2: ЧАСТОТА — локальный FFT ROI (zero-pad ×4) → argmax → ПАРАБОЛА 3 точек
        δ=½(P₋−P₊)/(P₋−2P₀+P₊) на log-мощности → f̂=(k+δ)·fs/N.
        Факт: ошибка 0.6 кГц на чистом 160-отсч ROI (бин был 15.6 МГц!).  ← напоминание Alex
Этап 3: ЧТО/КОГДА ТОЧНО — банк опор из WaveformFactory (реюз, формулы не дублируем):
        тип{radio,am} × длит{4,8,16п} × микросетка f̂±0.5 МГц × фазы огибающей (am, 4 шт).
        ρ=|corr|²/(σ̂²·‖ref‖²) ~ Exp(1) при H0 → порог −ln(pfa). Выигрыш 10·log10(D).
```

Ключевые пороги (математика ↔ наблюдение сходятся):
- рассогласование частоты: −3 дБ при δf ≈ 0.44·fs/D (D=160 → 1.4 МГц) ⇒ параболы хватает;
- **фаза огибающей АМ**: формула ядра считает envelope от абсолютного t ⇒ опора с t0=0
  НЕ совпадает по фазе с импульсом на t0=2900 — лечится сеткой 4 фаз (иначе тип врёт);
- σ̂² честно: median(|x|²)/ln2 (|x|²~Exp вне сигнала), реюз `estimate_noise_floor`.
