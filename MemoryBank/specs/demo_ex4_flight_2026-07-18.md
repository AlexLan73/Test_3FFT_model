## 3. Такт (конвейер)

```
для каждой движущейся сущности (цель, носитель гребёнки, barrage-угол):
    state ← случайная модель.propagate(state, dt)     # выбор модели на старте прогона, от seed
    sample ← Kinematics.project(state) → (kx, ky, r→t0 линейно в [400,3600])
volume ← эхо цели (S1) + гребёнка (серия копий за носителем) + barrage + шум
конвейер ex3: полоса → null → скан → VolumeTokenizer → детекции (+признаки §4.11)
tracker.update(...) → треки с №, is_moving
кадр компоновки §1.6 → в историю (для GIF ×2 и HTML)
```

Метрики: доля тактов с найденной целью/гребёнкой, ошибка угла, треки (№, длина, is_moving),
барьер: полоса детектирована/подавлена по тактам.
