# Производственные помехи: классификация для моделирования радарного пайплайна

**Документ:** Классификация и параметры помех для синтеза датасета  
**Контекст:** Каскадный 3D FFT-детектор, 1250 окон nx×ny (после pad N_pad_x×N_pad_y) по дальности  
**Дата:** 14.07.2026

---

## Обзор

Для правильного моделирования и обучения классификатора необходимо охватить весь зоопарк
промышленных помех, которые могут имитировать полезный сигнал или маскировать его.
Все источники разбиты по **трём главным осям классификации**: происхождение → форма
сигнала → частотный диапазон.

---

## 1. Классификация по происхождению

```text
Все помехи
├── Естественные (природные)
│   ├── Атмосферные (гроза, осадки)
│   └── Космические (солнечное излучение, ионосфера)
│
├── Искусственные непреднамеренные (индустриальные)
│   ├── Силовая электроника (VFD, IGBT, инверторы)
│   ├── Электрические разряды (дуга, корона, коммутация)
│   ├── Высоковольтные линии (ЛЭП)
│   ├── Освещение (люминесцентное, разрядное)
│   └── Цифровые системы (компьютеры, ПЛК, Wi-Fi)
│
└── Искусственные преднамеренные
    ├── Другие радары (взаимные помехи)
    ├── РЭП (заградительные, точечные, модулированные)
    └── Корреляционные помехи (ретранслятор)
```

## 2. Классификация по форме сигнала

| Тип | Описание | Проявление в nx×ny (после pad N_pad_x×N_pad_y) | Модель |
|-----|----------|--------------------|--------|
| **CW** | Узкополосный тон на фиксированной частоте | Один яркий пик в одном бине kx или ky | A·e^(j2πf₀t) |
| **NBI** | Полоса < 1 бина | Пик + несколько соседних бинов | Гауссова огибающая в частотной области |
| **PBI** | Полоса > 1, < полного диапазона | Несколько смежных бинов | Прямоугольная полоса в f-области |
| **BBI/заградительная** | Широкополосная, заполняет весь диапазон | Равномерно поднятый шумовой пол | N(0, σ²) с повышенным σ² |
| **Импульсная** | Короткие импульсы, высокая пиковая мощность | Всплеск энергии во всех бинах сразу | Δ-функция × случайные амплитуды |
| **ЛЧМ (sweep/chirp)** | Сигнал с линейной перестройкой | Диагональная полоса в матрице kx-ky | e^(j(2πf₀t + πμt²)) |
| **Структурный клаттер** | Стационарные отражатели | Постоянные пики в фиксированных бинах | Пики с нулевым доплером |
| **Мигающий клаттер** | Медленно флуктуирующие отражатели | Пики с малым доплером | Weibull или K-распределение |

|----------|---------|-------------------|---------------------|

| ОВЧ/УВЧ | 30–1000 МГц | Сварочная дуга, двигатели, щётки | Широкополосный шум, маскирует слабые цели |
| СВЧ L/S | 1–4 ГГц | Другие радары, Wi-Fi 2.4 ГГц | Прямая интерференция в полосе приёмника |

## 4. Главные группы производственных помех

### 4.1 Группа A: Силовая электроника VFD/IGBT ⚠⚠⚠

**Происхождение:** Преобразователи частоты (VFD) с ключами IGBT: насосы, вентиляторы,
краны, конвейеры, лифты.

**Механизм:** IGBT переключается с dV/dt ≈ 1–10 кВ/мкс. Несущая частота f_sw = 2–16 кГц
генерирует гармонический гребень:

```
f_n = n · f_sw,  n = 1, 2, 3, ...
```

**Характеристики спектра:**
- Проводимые помехи: 150 кГц — 30 МГц
- Излучаемые помехи: 30–500 МГц
- Форма: гармонический гребень + широкополосный шум между гармониками

**Параметры для модели:**
```
Тип:      периодическая гармоническая + широкополосная компонента
f_sw:     2–16 кГц (базовая)
Гармоники: до 100-й (до ~1.6 ГГц)
Амплитуда: -30 до -60 дБ относительно несущей радара
Форма в матрице nx×ny (после pad N_pad_x×N_pad_y): горизонтальная полоса или несколько точечных пиков
```

---

### 4.2 Группа B: Электрические разряды (дуга, корона) ⚠⚠⚠⚠

**Источники:** Сварочная дуга (MIG, TIG), корона на ЛЭП 110–750 кВ,
коммутация контакторов/реле.

**Характеристики TIG-сварки (измерения NIST 2025):**
- Диапазон: 900 МГц, 2.4 ГГц, 5.3 ГГц — все три полосы одновременно
- Ширина полосы: 160 МГц на канал
- Форма: квазислучайные импульсы с повторяемой структурой

**Характеристики коронного разряда:**
- Диапазон: от нескольких МГц до 1 ГГц
- Tричелевские импульсы: частота повторения 10 кГц – 10 МГц
- Спектр: широкополосный с характерными пиками

**Параметры для модели:**
```
Тип:     импульсный шум (Impulsive Noise)
Распределение амплитуд: α-stable (α ≈ 1.2–1.6) или смесь Гауссиан
Период повторения:     случайный, от мкс до мс
SNR сварочной дуги:   +5..+20 дБ выше теплового шума при 10–30 м
```

---

### 4.3 Группа C: Линии электропередач (ЛЭП) ⚠⚠

**Механизм:** Ток 50 Гц и гармоники 100, 150, … 5000 Гц наводятся на цепи приёмника.
В дождь/туман корона ЛЭП расширяется до 500 МГц, амплитуда растёт на 10–20 дБ.

**Параметры для модели:**
```
Тип:     аддитивный широкополосный шум с флуктуацией (log-normal amplitude)
Полоса:  0.15–500 МГц (основная энергия до 30 МГц)
Зависимость от погоды: амплитуда пропорциональна влажности
Пространственно: изотропный (все бины nx×ny равномерно)
```

---

### 4.4 Группа D: Электродвигатели ⚠⚠ / ⚠⚠⚠

**Щёточные DC-двигатели:** широкополосный шум 1 МГц–1 ГГц (искра коллектора).  
**Бесщёточные (BLDC):** кондуктивные гармоники по схеме VFD/IGBT (см. гр. A).  
**Прямой пуск:** только 50 Гц + 3-я, 5-я, 7-я гармоники.

**Параметры для модели:**
```
Тип:     смешанный: импульсный (щётки) + гармонический (VFD)
Полоса щёток: 1–1000 МГц, спад -50 дБ/декада
Зависимость от нагрузки: амплитуда пропорциональна моменту
```

---

### 4.5 Группа E: Взаимные помехи от других радаров ⚠⚠⚠⚠⚠

| Сценарий | Тип помехи | Форма в матрице nx×ny (после pad N_pad_x×N_pad_y) |
|----------|------------|----------------------|
| CW радар скорости | Узкополосная CW | Один острый пик — имитирует цель |
| Импульсный радар | Импульсные всплески | Случайные пики по всей матрице |
| FMCW другого диапазона | Ширококполосная sweep | Диагональная полоса |
| Метеорадар S/C-диапазон | Периодические импульсы | Строчная структура |

CW-сигнал соседнего радара поднимает шумовой пол на 5–10 дБ.

**Параметры для модели:**
```
CW-помеха:    A·sin(2π·f_int·t + φ_rand)
Импульсная:   прямоугольные импульсы τ=0.1–10 мкс, PRF=100–10000 Гц
FMCW-помеха:  chirp с μ ≠ μ_свой, перекрытие в ±30% полосы
```

### 4.6 Группа F: Клаттер (отражения от местности и среды) ⚠⚠ / ⚠⚠⚠

| Тип | Доплер | Статистика | Форма в nx×ny (после pad N_pad_x×N_pad_y) |
|-----|--------|------------|---------------|
| Наземный | ≈0 | Weibull, K-distribution | Пики в центральных бинах |
| Морской | ±2–10 уз | Weibull (α=1–2) | Размазанная кольцеобразная |
| Дождь | ±5–15 м/с | Гауссова, Rayleigh | Диффузный фон |
| Биоклаттер | ±5–15 м/с | Swerling I/II | Точечные цели |

---

### 4.7 Группа G: Цифровые/беспроводные системы ⚠⚠

Wi-Fi (2.4/5 ГГц) OFDM, полоса 20/40/80 МГц, период фрейма 10 мс → случайные
по времени всплески, не постоянные пики.

## 6. Математические модели для синтеза датасета

### 6.1 CW-помеха (INT_CW)

```
x_CW(t) = A_CW · cos(2π · f_int · t + φ₀)
```

В матрице nx×ny (после pad N_pad_x×N_pad_y) — один пик в бине (kx_int, ky_int).
Амплитуда A_CW: от порога шума до +20 дБ над ним.

### 6.2 Импульсная / Дуга (IMP_ARC)

```
x_imp(t) = Σ_k  A_k · δ(t - t_k) * h(t)
```

h(t) — экспоненциальный спад τ ≈ 0.1–1 мкс,
A_k ~ α-stable (α ≈ 1.4), t_k — пуассоновский поток.

### 6.3 VFD-гармоники (HAR_VFD)

```
x_VFD(t) = Σ_{n=1}^{N} A_n · cos(2π·n·f_sw·t + φ_n) + w_broad(t)
```

f_sw ∈ [2, 16] кГц, N = floor(f_max / f_sw).

### 6.4 Наземный клаттер (CLUT_GND)

Концентрируется в бинах kz ≈ 0:

```
C_gnd(kx, ky) = Σ_m  σ_m · δ(kx - kx_m, ky - ky_m) * G(kx, ky)
```

G — Гауссово размытие (апертурный эффект), σ_m ~ Weibull (c = 0.5–2.0).

### 6.5 Суммарная модель куба

```
X_total = X_target + X_thermal + X_VFD + X_IMP + X_CW + X_clutter
```

Каждая компонента включается с вероятностью p_i:

| Компонента | p_i |
|------------|-----|
| Тепловой шум | 1.0 (всегда) |
| VFD-гармоники | 0.30 |
| Импульс дуги | 0.15 |
| CW чужого радара | 0.10 |
| Наземный клаттер | 0.40 |
| FMCW-помеха | 0.05 |

## 7. Уязвимости пайплайна 1250 × nx×ny (N_pad_x×N_pad_y)

### Уровень 1 (энергетический CFAR)
- **IMP_ARC** → ложная тревога (всплеск E_n)
- **HAR_VFD** → поднимает µ_noise, снижает Pd
- **Нестационарность** → медленная адаптация порога

**Решение:** OS-CFAR или тест куртозиса вместо CA-CFAR.

### Уровень 2 (Conv3D N_pad_x×N_pad_y×32)
- **INT_CW** → один пик = ложная "цель" для MF и MLP
- **CLUT_GND** → постоянные пики в нулевых доплеровских бинах

**Решение:** обучение CNN на примерах с клаттером; AE на чистом шуме.

### Уровень 3 (Conv3D N_pad_x×N_pad_y×256)
- **INT_FMCW** → при длинном окне видна как диагональная "рельса"
- **Медленный клаттер** → длинное окно разделяет клаттер и медленную цель

## 9. Выводы

1. **Три критически опасных группы:** CW-помеха (имитатор цели), дуга (срыв CFAR), VFD
   (деградация шумового пола).
2. **VFD/IGBT** — самый распространённый в промышленной зоне: IGBT dV/dt ~ 1–10 кВ/мкс,
   гармоники до 1 ГГц.
3. **CW от соседнего радара** → один яркий пик в матрице nx×ny (после pad N_pad_x×N_pad_y), неотличимый от цели без
   анализа нестационарности.
4. **Наземный клаттер** → нулевые доплеровские бины, модель Weibull/K-distribution.
5. **Для Уровня 1** рекомендуется OS-CFAR + тест куртозиса против импульсных разрядов.
6. **Автоэнкодер Уровня 2** обучать на шуме + VFD-фоне без целей; дуга и CW будут
   "аномалиями" — система корректно поднимет сигнал тревоги.
