## 1. Что ТОЧНО зафиксировано (Alex подтвердил)

- **Ось:** N = **4096 отсчётов** (семплов), комплексные IQ.
- **АЦП:** fs = **500 МГц** (Найквист ±250 МГц).
- **Амплитуда** несущей/сигнала = **1**.
- **Размещение:** сигнал кладём в прямоугольном окне (`TimeWindow(kind="short")`), вне окна — нули.
- **Реюз генераторов:** только через `WaveformFactory().create(Modulation.*)`, формулы НЕ дублируем.
- **Примерная структура** Примерная структура файлов `MemoryBank\specs\demo_examples_2026-07-18.md`
- **Два типа сигнала** (оба нужны для тестов, ни один не удалять):
  - **A. Радиоимпульс** (`kind="radio"`) — несущая в окне, БЕЗ модуляции.
    Магнитуда `|a(t)|` = **прямоугольник**.
  - **B. АМ** (`kind="am"`) 
АМ = медленная огибающая на быстрой несущей.
Модулирующий: s(t) = cos(2π·F·t)
Несущая: c(t) = cos(2π·f₀·t)
АМ: u(t) = A·(1 + m·s(t))·cos(2π·f₀·t)
Раскрывая произведение:
u(t) = A·cos(2πf₀t) + (A·m/2)·cos(2π(f₀+F)t) + (A·m/2)·cos(2π(f₀−F)t)
— несущая + две боковые. Вся информация в боковых, несущая жрёт мощность впустую (отсюда DSB-SC и SSB).   
мы должны протестировать к разными несущеми частотами f_m0 = 250Мгц, f_m1 = 100Мгц, f_m2 = 50Мгц       
  - — огибающая частоты `f_ogib`  - (см. §2). Магнитуда = **волна (горбы)**.
  - f_ogib - считаем из формулы f_m - 8 периодов = 1 период - это базовая частота огибающей на этот експеремент
  - длинна импульса 1. f_m - 4 периода; 2. f_m - 8 периодов; 3. f_m - 16 периодов
