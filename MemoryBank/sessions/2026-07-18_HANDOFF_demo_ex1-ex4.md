## Ключевые решения/находки (не потерять)

1. **Эхо S1** — `core/generators/waveforms/waveform_to_cube.py::build_pulse_echo_volume`:
   задержанная копия зонда, огибающая АМ с фронта (|фронт|=1+m). Прецедент — build_lfm_target_volume.
2. **Полосовая помеха ломает эскалацию** → конвейер: скан → полоса (столбец всплесков)
   → гейт → **rank-1 null** (известный угол; EVD/MVDR на M=4096 непрактичны — в докстрине ex3)
   → повторный скан. Сверка с SubspaceNuller в тестах (>60 дБ).
3. **Скорость**: полный `VolumeTokenizer` — python-CFAR, НЕ гонять на каждом такте/окне.
   Сопровождение по грубой карте (патент), признаки — точечно. 17 мин → 3 мин (ex4).
4. **coarse_burst_points(max_points=)**: порог max-статистики `T=N̂·ln(n/pfa)` + NMS до K
   пиков на окно (один argmax терял цель рядом с сильной гребёнкой).
5. **Общая графика**: `core/graphics/scene_points.py` (ScenePointsVisualizer, дальность по
   горизонту `AxisLayout.range_in_depth`) — сцены НЕ рисовать руками.
