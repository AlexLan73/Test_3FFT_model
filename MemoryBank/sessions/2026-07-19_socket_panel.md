# Сессия 2026-07-19 — web-панель полёта ex4 НА СОКЕТАХ (P6)

## Сделано

Переведена web-панель полёта ex4 с самодостаточного HTML на **сокеты** (правило 07 + handoff).

- **Сервер** `demo/ex4_flight/server.py`: `Ex4Flight.run_history` (R5, один прогон) → стрим тактов
  `meta` (апертура/камера/станции/финал-признаки) + `tick` (истина/точки/треки/срезы, МИРОВЫЕ
  координаты) через `core.runtime.WebSocketTransport`, порт 8765, дефолт 90 тактов.
- **Клиент** `web/`: `index.html` (канон §1.6) + `app.js` (canvas-рендер, WS-подписка) +
  `msgpack.js` (свой мини-декодер под схему `core/runtime/codec.py`, ноль внешних URL — офлайн).
  Проекция мир→экран на клиенте единой `core.graphics.Projection` (`as_js()`), вращение мышью локально.
- Старый самодостаточный `demo/ex4_flight/web.py` → `архив/ex4_web_selfcontained_2026-07-19.zip`, удалён из дерева.
